# MacroVox v1.0.0

## Quick Start
1. Run `MacroVox.exe`
2. Click tags to categorize your memo
3. Click REC to start recording
4. Click STOP to save

## Configuration
Edit `config.json` to customize:
- Output folder
- Microphone
- Audio format
- Tags and colors
- Theme (dark/light)

## Files
Recordings are saved as: `YYYYMMDD_HHMMSS_TAGS.wav`
